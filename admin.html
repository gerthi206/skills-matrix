Email Notifications (EmailJS)

SkillsBase uses EmailJS to send invitation emails directly from your browser — no server required. Create a free account →
Free plan: 200 emails/month. Takes ~3 minutes to set up.
📋 How to set up EmailJS (3 steps)
  1. Create account at emailjs.com → go to Email Services → connect Gmail, Outlook, or any SMTP.
  2. Create a template in Email Templates. Use these variables in your template body:
    {{to_name}} {{to_email}} {{temp_password}} {{role}} {{login_url}} {{from_name}}
  3. Copy your keys from Account → API Keys (Public Key) and from your Service/Template pages, paste them below.
